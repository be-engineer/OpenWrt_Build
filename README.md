## 说明
* ImmortalWrt Release 
* Linux-kernel: 5.15.x
* ipv4: 192.168.10.1
* 个人吃鹅固件
* username: root
* password: 空 / password
* 原汁原味非杂交! 感谢Immortalwrt/R2S Club/R4S Club/天灵/GC/QC等诸多大佬的努力!
* 支持scp和sftp
* 原生OP内置升级可用,固件重置可用
